# MessUpTetris

## 프로젝트 개요
- **장르**: 액션 아케이드
- **컨셉**: 퍼즐 기반 보스 배틀
- **개발 기간**: 2025.08 ~ 2025.10
- **팀 구성**: 4인 팀 프로젝트

## 전투 시스템
- 플레이어는 **망치를 이용한 단일 타격 공격** 수행
- **일반 블록**: 2회 타격 시 파괴
- **공격 블록**: 1회 타격 시 보스 방향으로 튕겨나가 공격

## 게임 플레이 구조
- **웨이브 기반 진행**
  - 각 웨이브 클리어 시 다음 단계로 이동
  - 웨이브마다 배경 변화
- **게임 오버 조건**
  - 블록이 일정 높이 이상 쌓일 경우 종료
- **보스 처치 시**
  - 해당 웨이브의 블록 초기화
